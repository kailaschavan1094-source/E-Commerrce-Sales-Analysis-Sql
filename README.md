# E-Commerce Sales Analysis using SQL

## Project Overview

This project analyzes an E-Commerce Orders dataset using SQL to generate business insights. The analysis focuses on customer behavior, product performance, sales trends, payment methods, coupon usage, referral sources, and revenue analysis.

---

## Dataset

- E-Commerce Orders Dataset
- Total Records: 1200
